Для того, чтобы начать игру, нажмите "Enter", чтобы выйти "Escape"

управление(кнопки): a, d, space, escape
для дебага сделан выход из fulscreen: alt+tab, а обратный вход: tab+alt

сервер выключается через минуту, если на нём никто не заспавнен
